一键安装指南

  • 打开 Windows PowerShell（建议以管理员身份运行）。
  • 复制并粘贴以下命令，然后按回车执行。
irm 'http://localhost:8080/scripts/install.ps1'|iex

如何卸载： 再次运行相同的脚本，但在环境变量中设置卸载动作或添加 uninstall 参数。

$env:RC_ACTION='uninstall';irm 'http://localhost:8080/scripts/install.ps1'|iex
  • 打开 终端 (Terminal) 应用程序。
  • 复制并粘贴以下命令，然后按回车执行。脚本会自动识别系统架构。
curl -kfsSL http://localhost:8080/scripts/install.sh | bash

如何卸载：使用相同命令，但在后面加上 -s -- uninstall 参数。

curl -kfsSL http://localhost:8080/scripts/install.sh | bash -s -- uninstall
